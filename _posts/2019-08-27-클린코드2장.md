## 클린코드 - 의미 있는 이름



### 의도를 분명히 밝혀라

```java
// 잘못된 예
int d; // 경과 시간

// 올바른 예
int elapsedTimeInDays;
int daysSinceCreation;
```

<br/><br/>

### 클래스 이름

명사나 명사구가 적합하다.

- 올바른 예
  - Customer, WikiPage, Account, AddressParser
- 잘못된 예
  - Manager, Processor, Data, Info ..

<br/><br/>

### 메서드 이름

동사나 동사구가 적합하다.

- deletePage, save ...

<br/>

<br/>

### 의미 있는 맥락을 추가

firstName, lastName, city 대신

addrFirstName, addrLastName, addrCity
